## ğŸ§  How It Works

ShieldPass evaluates passwords based on:

- Length (8+ characters)
- Mix of UPPER and lower case letters
- Inclusion of digits
- Special characters (!@#$%^&*)

It returns a score from **0 to 4**, giving feedback and security tips to help users optimize their credentials.
